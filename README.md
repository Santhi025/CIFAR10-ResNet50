# CIFAR-10 Image Classification using ResNet50

## Overview

This project implements a ResNet50-based image classifier on the CIFAR-10 dataset using TensorFlow/Keras.

## Dataset

- CIFAR-10
- 60,000 images
- 10 object classes

## Model

- ResNet50
- Transfer Learning
- TensorFlow/Keras

## Results

- Test Accuracy: 93.73%
- Weighted F1-Score: 0.94
